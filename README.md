# Одесская Генеалогия

Сайт для хранения и поиска родословных — с фокусом на Одессу.

---

## Что умеет сайт

- Поиск по фамилии, имени, отчеству и месту рождения
- Страница каждого человека с его данными и родственниками
- Защищённая паролем админ-панель для добавления записей и связей
- Работает онлайн, доступен всем

---

## Шаг 1 — Установка Python и VS Code

1. Скачай Python: https://python.org/downloads (выбери Windows/Mac)
   - При установке ОБЯЗАТЕЛЬНО отметь галочку "Add Python to PATH"
2. Скачай VS Code: https://code.visualstudio.com
3. В VS Code установи расширение "Python" (автор Microsoft)

---

## Шаг 2 — Запуск локально (на своём компьютере)

Открой папку `genealogy` в VS Code, затем открой терминал (Ctrl+\`) и выполни:

```bash
# Установи зависимости
pip install -r requirements.txt

# Запусти сервер
python app.py
```

Открой браузер: http://localhost:5000

**Пароль администратора по умолчанию:** `odessa1794`
(Поменяй в app.py строку `ADMIN_PASSWORD` или задай переменную окружения)

---

## Шаг 3 — Деплой на Render.com (бесплатный хостинг)

### 3.1 Загрузи код на GitHub
1. Зарегистрируйся на https://github.com
2. Создай новый репозиторий (New repository), назови его `genealogy`
3. Скачай GitHub Desktop: https://desktop.github.com
4. В GitHub Desktop: File → Add Local Repository → выбери папку `genealogy`
5. Нажми "Publish repository"

### 3.2 Задеплой на Render
1. Зарегистрируйся на https://render.com (можно через GitHub)
2. New → Web Service → подключи свой репозиторий `genealogy`
3. Настройки:
   - **Build Command:** `pip install -r requirements.txt`
   - **Start Command:** `gunicorn app:app`
   - **Environment:** Python 3
4. Добавь базу данных: New → PostgreSQL (бесплатный план)
5. В настройках Web Service добавь переменные окружения:
   - `DATABASE_URL` = строка подключения из PostgreSQL (Render сам подскажет)
   - `ADMIN_PASSWORD` = твой пароль (любой, который хочешь)
6. Нажми Deploy — через 2-3 минуты сайт готов!

## Как пользоваться сайтом

### Добавление людей
1. Зайди на `/admin/login` и введи пароль
2. В форме "Добавить запись" заполни данные человека
3. Сохрани — человек появится в базе

### Добавление родственных связей
1. В форме "Добавить связь" выбери двух людей и тип связи
2. Например: Иванов Иван → **отец** → Иванов Пётр

### Поиск
На главной странице просто начни вводить фамилию — результаты появятся мгновенно.

## Изменение пароля

В файле `app.py` найди строку:
```python
ADMIN_PASSWORD = os.environ.get('ADMIN_PASSWORD', 'odessa1794')
```
Замени `odessa1794` на свой пароль. Или задай переменную окружения `ADMIN_PASSWORD` на Render.

## Структура файлов

```
genealogy/
├── app.py              ← главный файл (Python/Flask)
├── requirements.txt    ← список библиотек
├── Procfile            ← инструкция для хостинга
└── templates/
    ├── index.html      ← главная страница (поиск)
    ├── person.html     ← страница человека
    ├── admin.html      ← админ-панель
    └── login.html      ← страница входа
```
